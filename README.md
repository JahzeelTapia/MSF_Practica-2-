Objetivo.
Diseñar un controlador que permita formular un protocolo de tratamiento para que un paciente
con enfisema (caso) presente la misma presión alveolar que un individuo sano (control).

Actividades
Calcular analíticamente la función de transferencia del sistema.
Establecer el modelo de ecuaciones integro-diferenciales.
Determine el error en estado estacionario y la estabilidad del sistema en lazo abierto.
Diseñar el controlador con Simulink utilizando el bloque PID Controller y la herramienta Tune para sintonizar los valores óptimos para cada una de las ganancias kP, kI y kD.
Determinar la respuesta al caso y control en Python, Simulink y Multisim del circuito RLC.
